# 宝可梦图鉴应用


# 分为离线版本和在线版本
## 功能需求

### 1. 搜索调用宝可梦信息库（本地下载）
- **后端搜索名字**：用户通过输入宝可梦的名字进行搜索。  
- **图片识别技术**：支持通过上传宝可梦的图片来识别并查找对应的信息。  
- **返回至前端**：  
  - **展示内容**：包括宝可梦的图片、属性、介绍、种族值及分类等详细信息。
  - **解锁玩法**：待定

### 2. 图片识别输出识别结果
- 用户上传一张宝可梦的图片，系统通过图像识别技术确定宝可梦的身份，并返回相应的信息。

### 3. 语音播报
- 提供语音功能，能够将选中的宝可梦信息通过语音播放出来，增加互动性。

### 4. 加入用户信息
- 支持用户注册登录，允许用户：
  - 保存喜欢的宝可梦信息；（点击爱心喜欢）
  - 享受个性化功能。

## 具体实现方式

### 数据组织
- **每个宝可梦为一页**：所有宝可梦的信息将以独立页面的形式呈现，方便用户浏览和查询。  
- **列表 `{内容}` 储存**：每个宝可梦的数据（如名称、图片 URL、描述等）都以结构化列表形式存储，便于快速检索和展示。

### 技术实现细节
- **联网服务**
- **后端服务**：负责处理搜索请求（包括文字搜索和图片识别），并从本地数据库中提取相应数据。  
- **前端界面**：设计直观的 UI 界面，用于显示搜索结果以及提供图片上传入口。  
- **图像识别模块**：集成先进的图像识别 API 或模型，用于解析用户上传的宝可梦图片。  
- **语音播报功能**：利用文本转语音（TTS）技术，为用户提供听觉上的信息获取途径。  
- **用户管理系统**：构建安全可靠的用户认证体系，确保用户信息的安全性和隐私保护。
